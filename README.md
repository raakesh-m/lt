# Legatia Tech Website

A modern, responsive website for Legatia Tech built with Next.js 14, Tailwind CSS, and TypeScript.

## Features

- Modern UI/UX
- Responsive Design
- Dark/Light Mode
- Fast Performance
- SEO Optimized

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Shadcn UI
- Framer Motion
